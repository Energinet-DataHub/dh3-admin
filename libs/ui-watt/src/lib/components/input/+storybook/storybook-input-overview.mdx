## How to use
Import the `WattFormFieldModule` and `WattInputModule` module
```typescript
import { WattFormFieldModule } from '@energinet-datahub/watt';
import { WattInputModule } from '@energinet-datahub/watt';

@NgModule({
  imports: [
    WattFormFieldModule,
    WattInputModule
  ]
}) class YourModule {}
```
***NOTE:** Remember also to import `FormsModule` and `ReactiveFormsModule` from Angular.
