# Primary

Contained buttons are high-emphasis, distinguished by their use of fill.
They contain actions that are primary to your app.

# Secondary

Outlined buttons are medium-emphasis buttons.
They contain actions that are important, but aren't the primary action in an app.

<Story name="Secondary">
  {{

        <watt-button variant="secondary">Enabled</watt-button>
        <watt-button variant="secondary" [disabled]="true">Disabled</watt-button>
        <watt-button variant="secondary" [loading]="true">Hidden</watt-button>
        <watt-button variant="secondary" icon="plus">Enabled</watt-button>
        <watt-button variant="secondary" icon="plus" [disabled]="true">Disabled</watt-button>

  }}
</Story>
